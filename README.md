# Helluva Calculator

Prosty kalkulator napisany w Pythonie z użyciem biblioteki Pygame.
Projekt zawiera podstawowe działania matematyczne oraz prosty interfejs graficzny z przyciskami.

## Funkcje

* Dodawanie
* Odejmowanie
* Mnożenie
* Dzielenie
* Obsługa liczb dziesiętnych
* Czyszczenie wyniku (`C`)

## Technologie

* Python
* Pygame
